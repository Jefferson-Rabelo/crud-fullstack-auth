# CRUD Fullstack com Autenticação JWT

## 🚀 Tecnologias
- React
- Tailwind CSS
- Node.js
- Express
- MySQL
- JWT
- bcrypt
- Axios

## 📦 Funcionalidades
- Cadastro de usuários
- Login
- JWT Authentication
- Rotas protegidas
- CRUD completo

## ▶️ Como rodar

### Backend
```bash
npm install
npm run dev
```

### Frontend
```bash
npm install
npm run dev
```

## 🔐 Variáveis ambiente

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
JWT_SECRET=
```
